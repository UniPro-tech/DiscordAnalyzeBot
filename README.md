# Discord Analyze Bot

Discord上のさまざまなアクティビティを記録し、分析するBot。

## 使い方(Self-host)

1. `_docker-compose.yaml`を`docker-compose.yaml`にコピーします。
2. `docker-compose.yaml`の環境変数を編集します。
